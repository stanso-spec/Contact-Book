## Description
A simple command-line Contact Book developed in Python that allows users to store and manage contact information. The application enables users to add new contacts, view all saved contacts, search contacts by name, and delete contacts through an easy-to-use menu-driven interface.

## Features
• Add new contacts  
• View all contacts  
• Search contacts by name  
• Delete existing contacts  
• Menu-driven CLI interface  

## Technologies Used
• Python 3  
• Lists and Dictionaries  
• Loops and Conditional Statements  

## How to Run

1. Run the program:
   python main_3.py
